Example 1: Guided Practice

In this first example, we will show you step-by-step how to use a table to solve a system of equations.

[ Video ]

Now that we are ready to solve a system of equations using tables, we will create the table for two equations the same way that we created a table for a single equation. Let’s find all of the y-values for the y sub 1 =-3x first. If we replace x with -2 we get the equation y sub 1 = -3(-2). We simply multiply to get y sub 1 = 6. We place that number into the table and move on to the next x-value. If we substitute -1 for x in the first equation we get y sub 1 = -3(-1). We multiply to get y sub 1 = 3. Next we plug in 0. Y sub 1 = -3(0), so y=0. If we plug in 1, we have y sub 1 =-3(1), y=-3. Lastly, we plug in 2 so, y sub 1 = -3(2), so y=-6. Let’s do the same thing with the second equation. Plug in -2 for x so we get y sub 2 =-2+4, if we add we get y sub 2 =2. Let’s put that in the table and move on to x=-1. Plug that in to the equation to get y sub 2 = -1+4, so y = 3. If we plug in 0 we get y sub 2 = 0+4, so y=4. Plugging in 1 we get y sub 2=1+4, y=5. Lastly, we plug in 2 for x to get y sub 2 =2+4, so then y=6. Now if we look at our completed table we can see when x=-1 both y sub1 and y sub2 are equal to 3. This is the solution to our system of equations. We write our answer as a coordinate point. (-1,3).
