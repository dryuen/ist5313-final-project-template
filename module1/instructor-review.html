Let's review tables!

[ Video ]

Let's start by reviewing how to create a table from an equation. Given the equation y = -2x + 4 the first thing we should do is replace the x in the equation with the first x-value from the table. We write y = -2 times -2 + 4. Our order of operations says we have to multiply before we add, so let's start by multiplying the -2 and the -2. We get y = +4 plus 4. Last thing we need to do is add the 4s. So y = 8. Now I can fill that into my table.

Okay this time we're going to do the same process but we're going to use x is equal to -1. So y = -2 times -1 plus 4. Again we multiply first. We get y = +2 plus 4. And lastly we add to get y = 6. Then we fill it into the table.

We continue this with 0 so y = -2 times 0 plus 4. Multiply we get 0 + 4 and add those to get y = 4.

For this next one we're going to replace x with a 1. y = -2 times 1 plus 4. We multiply to get -2 plus 4, and when we add those we will get +2. All right our last one we replace the x with a positive 2. We get y = -4 + 4. Add those together and we get 0. So there's our completed table from the equation y = -2x + 4.
