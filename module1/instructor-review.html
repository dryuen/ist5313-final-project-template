Let's review tables!

[ Video ]

We'll start by reviewing how to create a table from an equation. Given the equation y=-2x+4 the first thing we should do is replace the x in the equation with the first value from the table. We write y=-2(-2)+4. Our order of operations says that we have to multiply before we add, so if we multiply -2 times -2 we get positive 4. Our new equation reads y=4+4. So, y=8. Now we can fill this in to the table. Letâ€™s continue this process for the rest of the x-values in the table. If we substitute -1 for x we get y=-2(-1)+4. We multiply -2 and -1 to get positive 2. So now we have y=2+4 which gives us 6. We fill that in to the table and keep going. y=-2(0)+4. Multiply to get y=0+4, so y=4. y=-2(1)+4. Multiply to get y=-2+4, so y=2. Last one, y=-2(2)+4. We multiply and get y=-4+4, so y=0. And now our table is complete.
