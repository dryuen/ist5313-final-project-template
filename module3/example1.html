Example 1: Guided Practice

In this first example, we will show you step-by-step how to use elimination to solve a system of equations.

[ Video ]

When you have two equations that are in standard form instead of slope-intercept form it is often easier to solve using elimination. To start solving you must be sure that the two equations are lined up, x above x, y above y, and constant above constant. Our goal is to be able to add the equations straight down and have one of the variables cancel out. If we were to multiply the top equation by -2, then if we add the two equations the x terms would cancel out. Let’s try.

First put a parenthesis around the entire top equation. We must multiply every term by -2. After distributing we get -2x-6y=-22. I’ll rewrite the second equation below. Now we add the two equations straight down. -2x+2x is 0, so it cancels out and we don’t have to write it. -6y+-4y = -10y. Bring down the equals sign. -22+-18 =-40. Now we have a one-step equation to solve. We can simply divide both sides by -10 to get y=4. Now that we know the value of y we can find the value of x by plugging in the y-value into one of the equations. I am going to substitute in place of y in the first equation. That gives me x+3(4)=11.

Now we can solve this equation. Multiplying first gives us x+12=11. Now we can subtract 12 from both sides, so x=-1. To write our final answer we plug in the x-value and y-value into a coordinate point. So the solution to this system of equations is (-1,4).
